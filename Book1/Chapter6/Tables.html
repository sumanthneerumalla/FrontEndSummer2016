Table headings

The 'th' element is used like the 'td' element except its used to represent the headings for columns or rows. Its important to use empty 'th' elements in order to represent empty cells otherwise the table wont display properly. The scope attribute is used to determine whether the heading is for a row or a column. Since i'd just like to move on from this chapter i'll just be copying a table from the textbook.

Spanning Columns and Rows

Sometimes entries in a table need to be stretched acros more than one column or row. The colspan and rowspand attributes can be used on th or td elements to indicate how many columns or rows it needs to run across.
