Email links

Its possible to create mail links by using the a tag and an href atribute. In order to make it different from a regular link however, you have to start the href attribute with a "mailto:". You can try clicking the following link to email github support: github support.Be forewarned that I haven't verified whether that is a valid email. It was only meant to illustrate a point.

Opening links in a new window

You can open a link ina new window by adding another attribute to the 'a' element. This link opens the Links2 page in a new window.

Linking to a specific part in the same page

Its also possible to link to specific parts in the same page. The following links take you different sections of this webpage, in no particular order

  • Absolute URLs
  • Email Links
  • Relative URLs
Keep in mind that it is possible to link to parts of another page by appending the '#' and the id of the section of another webpage to the end of the URL to the page. For example a link to the bottom of the homepage of the book I'm learning from would be "http://www.htmlandcssbook.com#bottom"
