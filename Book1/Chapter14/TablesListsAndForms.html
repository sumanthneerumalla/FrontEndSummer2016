Bullet Point Styles

The 'list-style-type' property allows you to control the shape or style of a bullet point. It can only be used on 'ol', 'ul', and 'li' elements and can be set eaual to

  • 'none;
  • 'disk'
  • 'circle'
  • 'square'
For ordered lists, the valid values are:
  • 'decimal'
  • 'decimal-leading-zero'
  • 'lower-alpha'
  • 'upper-alpha'
  • 'lower-roman'
  • 'upper-roman'
  • And more, even hebrew and armenian

Images for bullets

Images can be used in place of bullet points in unordered lists with the 'list-style-image: url()' property.

  • Ophelia
  • To Music
  • A Dream for Winter
  • Vowels
  • The Drunken Boat

Shorthand

All of the previous properties can be listed in shorthand with the 'list-style' property

Table Properties

Many of the previously used properties from boxes can be applied to table elements in order to make them more readable. Its suggested to give cells padding, distinguish headings by making them bolder or underlining them, aligning numerals to the right, shading alternate rows, and more
