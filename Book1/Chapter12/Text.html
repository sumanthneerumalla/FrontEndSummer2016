Specifying Typefaces

The font family property allows you to specify the typeace that should be used for any text inside the element to which a CSS rule applies. Typically the user has to have the specified typeface installed but there are way to import unavailable typefaces from within the CSS document itself. Its good practice to end with a generic font name for the type of font you would like. If a font name is made up of more than one word it should be put in double quotes. It is also good practice to not use more than 3 typefaces on a singe page

@font-face { font-family: 'ChunkFiveRegular'; src: url('fonts/chunkfive.eot');} h1, h2 { font-family: ChunkFiveRegular, Georgia, serif;}

Font Size

There are three different ways to specify the size of a font

  • Pixels: The number of of pixels followed by the letters px. Example: "font-size:12px;"
  • Percentages: The default size of text is 16px. Using "font-size: 200%; will result in a font size of 32px. Using 'font-size: 150%;' on an element that is a child of that element will result in a font of 48px
  • Ems: An 'em' is equivalent to the width of a letter m.

Importing Fonts

Using "@font-face" in Css allows you to use a font that is not on the users computer. The 'font-family' property specifies the name of the fotn that will be used in the rest of the style sheet. The 'src'property specifies the path to the font. The 'format' property specifies the format that the font is supplied in.Different browsers support different formats for fonts so different formats must be supplied to reach all browsers

Browser Compatibility Issues are a Pain

@font-face { font-family: 'ChunkFiveRegular'; src: url('fonts/chunkfive.eot'); src: url('fonts/chunkfive.eot?#iefix') format('embedded-opentype'), url('fonts/chunkfive.woff') format('woff'), url('fonts/chunkfive.ttf') format('truetype'), url('fonts/chunkfive.svg#ChunkFiveRegular') format('svg');} h1, h2 { font-family: ChunkFiveRegular, Georgia, serif;}

Font-Weight and Font-Style

The 'font-weight' and 'font-style' properties allow you to make certain elements to be 'normal'/'bold' and 'normal'/'italic'/'oblique

.credits { font-weight: bold; font-style: italic;} .title { font-weight: normal; font-style: italic;}

Upper and Lower Case

The 'text-transform' element is used to change the text of an aelement to 'uppercase', 'lowercase', or 'capitalize'.Setting the 'text-transform' element to 'capitalize' capitalizes the first letter of each word

Underline&Strike

The 'text-decoration'can be set to

  • 'none': Removes any decoration already applied to the text
  • 'underline':Underlines the text

  • 'overline':Adds a line over the top of the text

  • 'line-through':adds a line through the text

  • 'blink':Makes the text blink

Leading

'line-height' is an property that is used to set the vertical space between the lines of text. Pronounced "ledding", it is defined by typographers as the distance between the highest part of the lower line of text and the lowest part of the higher line of text. It is good practice to set the value in 'em' so that users can adjust the default size of their text in their browser and have the value of 'line-height' scale automatically.You may have noticed that this paragraph is set to a value of 2 ems Interesting Fact: The difference between the font-size and line-height is equal to the leading.
