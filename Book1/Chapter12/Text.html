Specifying Typefaces

The font family property allows you to specify the typeace that should be used for any text inside the element to which a CSS rule applies. Typically the user has to have the specified typeface installed but there are way to import unavailable typefaces from within the CSS document itself. Its good practice to end with a generic font name for the type of font you would like. If a font name is made up of more than one word it should be put in double quotes. It is also good practice to not use more than 3 typefaces on a singe page

@font-face { font-family: 'ChunkFiveRegular'; src: url('fonts/chunkfive.eot');} h1, h2 { font-family: ChunkFiveRegular, Georgia, serif;}

Font Size

There are three different ways to specify the size of a font

  • Pixels: The number of of pixels followed by the letters px. Example: "font-size:12px;"
  • Percentages: The default size of text is 16px. Using "font-size: 200%; will result in a font size of 32px. Using 'font-size: 150%;' on an element that is a child of that element will result in a font of 48px
  • Ems: An 'em' is equivalent to the width of a letter m.

Importing Fonts

Using "@font-face" in Css allows you to use a font that is not on the users computer. The 'font-family' property specifies the name of the fotn that will be used in the rest of the style sheet. The 'src'property specifies the path to the font. The 'format' property specifies the format that the font is supplied in.Different browsers support different formats for fonts so different formats must be supplied to reach all browsers

Browser Compatibility Issues are a Pain

@font-face { font-family: 'ChunkFiveRegular'; src: url('fonts/chunkfive.eot'); src: url('fonts/chunkfive.eot?#iefix') format('embedded-opentype'), url('fonts/chunkfive.woff') format('woff'), url('fonts/chunkfive.ttf') format('truetype'), url('fonts/chunkfive.svg#ChunkFiveRegular') format('svg');} h1, h2 { font-family: ChunkFiveRegular, Georgia, serif;}

Font-Weight and Font-Style

The 'font-weight' and 'font-style' properties allow you to make certain elements to be 'normal'/'bold' and 'normal'/'italic'/'oblique

.credits { font-weight: bold; font-style: italic;} .title { font-weight: normal; font-style: italic;}

Upper and Lower Case

The 'text-transform' element is used to change the text of an aelement to 'uppercase', 'lowercase', or 'capitalize'.Setting the 'text-transform' element to 'capitalize' capitalizes the first letter of each word

Underline&Strike

The 'text-decoration'can be set to

  • 'none': Removes any decoration already applied to the text
  • 'underline':Underlines the text

  • 'overline':Adds a line over the top of the text

  • 'line-through':adds a line through the text

  • 'blink':Makes the text blink

Leading

'line-height' is an property that is used to set the vertical space between the lines of text. Pronounced "ledding", it is defined by typographers as the distance between the highest part of the lower line of text and the lowest part of the higher line of text. It is good practice to set the value in 'em' so that users can adjust the default size of their text in their browser and have the value of 'line-height' scale automatically.You may have noticed that this paragraph is set to a value of 2 ems Interesting Fact: The difference between the font-size and line-height is equal to the leading.

Letter & Word Spacing

Kerning is the term for the space between ech letter. This can be controlled with the 'letter-spacing' element. The 'word-spacing' element can also be used to change the spacing between words. By default word spcing is set to 0.25em.

Alignment

The 'text-align' property allows you to control the alignment of text. It can be set to

  • 'left' to align text to left side
  • 'right' to alight text to the right side
  • 'center' to center the text
  • 'justify' to make every line but the last take up the full width of the containing box.

Vertical Alignment

The 'vertical-align' property is typically used to align text with inline elements such as 'em', or 'img', or 'strong'

Indenting Text

The 'text-indent' property allows you to indent the first line of text within an element. The amount to indent is usually doen in px or em.

  • Text can be indented to the right with positive values
  • Text can also be indented to the left
  • So much so that the text can go off screen

CSS3 Drop Shadows

The 'text-shadow' property is used to create a drop shadow behind words. It can also be used to create an embossed effect by adding a shadow that is slightly lighter than the text. There are four values for this property:

  1. The first value specifies how far left or right the shadow should fall
  2. The second value specifies how far up or down the shadow should fall
  3. The third value is optoinal, but determines the amount of blur that should be applied to the shadow
  4. The fourth value is the color of the drop shadow

Sample CSS

#one { background-color: #eeeeee; color: #666666; text-shadow: 1px 1px 0px #000000;}
#two { background-color: #dddddd; color: #666666; text-shadow: 1px 1px 3px #666666;}
#three { background-color: #cccccc; color: red; text-shadow: 1px 30px 7px red;}
#four { background-color: #bbbbbb; color: #cccccc; text-shadow: -1px -2px #666666;}
#five { background-color: #aaaaaa; color: #ffffff; text-shadow: -1px -1px #666666;

First Letter or Line

The ':first-letter' and ':first-line' properties are pseudo-elements because they come directly afterthe end of the CSS selector

CSS Source

p.intro:first-letter { font-size: 200%;} p.intro:first-line { font-weight: bold;}

Styling Links

The ':link' and ':visited' are considered pseudo-classes that allow you to set styles for links that have and have not been visited yet.

a:link { color: deeppink; text-decoration: none;} a:visited { color: black;} a:hover { color: deeppink; text-decoration: underline;} a:active { color: darkcyan;}

Responding to users with :hover, :active, and :focus

The :hover, :active, and :focus pseudo-classes are used to change the appearance of elements when a user is interacting with them. :hover applies when the mouse is above the element.: active is for when the user clicks the element. This is often used to make it look like the button is actually being pressed when the user presses down on their mouse.It should be kept in mind that the pseudo classes must always appear in the following order:

  1. :link
  2. :visited
  3. :hover
  4. :focus
  5. :active

CSS Source

input { padding: 6px 12px 6px 12px; border: 1px solid #665544; color: #ffffff;} input.submit:hover { background-color: #665544;} input.submit:active { background-color: chocolate;} input.text { color: #cccccc;} input.text:focus { color:#665544;}
