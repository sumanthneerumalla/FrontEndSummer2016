Specifying Typefaces

The font family property allows you to specify the typeace that should be used for any text inside the element to which a CSS rule applies. Typically the user has to have the specified typeface installed but there are way to import unavailable typefaces from within the CSS document itself. Its good practice to end with a generic font name for the type of font you would like. If a font name is made up of more than one word it should be put in double quotes. It is also good practice to not use more than 3 typefaces on a singe page

Font Size

There are three different ways to specify the size of a font

  • Pixels: The number of of pixels followed by the letters px. Example: "font-size:12px;"
  • Percentages: The default size of text is 16px. Using "font-size: 200%; will result in a font size of 32px. Using 'font-size: 150%;' on an element that is a child of that element will result in a font of 48px
  • Ems: An 'em' is equivalent to the width of a letter m.
